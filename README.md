# ionic3-angular4-charts-example

## just need few tools and modules:
- Node.js
- Ionic 3 and Angular 4
- Angular 2 Charts
- Charts.js

To run locally:
- Clone this repository
- Run 'npm install'
- Run 'ionic serve --lab'
